# Controlei — Frontend Documentation

## Overview

O frontend do **Controlei** é responsável pela interface do usuário do sistema de controle financeiro.

Ele permite que usuários:

- Criem contas
- Façam login
- Registrem receitas, despesas e investimentos
- Visualizem dados financeiros no dashboard
- Definam metas financeiras

O frontend é construído utilizando:

- React
- Vite
- JavaScript
- CSS

## Integração

O frontend consome a API do backend responsável por:

- autenticação
- gerenciamento financeiro
- metas
- relatórios
