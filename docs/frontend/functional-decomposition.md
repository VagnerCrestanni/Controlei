# Decomposição Funcional

O sistema frontend do Controlei é dividido nos seguintes módulos.

## 1. Autenticação

Responsável por:

- login
- criação de conta
- recuperação de senha

Componentes:

- Login
- CreateAccount
- ForgotPassword

---

## 2. Dashboard

Responsável por apresentar:

- resumo financeiro
- metas
- gráficos

Componentes:

- Dashboard
- CardDashboard
- Metas

---

## 3. Controle Financeiro

Responsável por:

- registrar transações
- visualizar transações

Componentes:

- ControleFinanceiro
- CardFinancialControl

---

## 4. Navegação

Responsável pela navegação da aplicação.

Componentes:

- Sidebar
- Navbar
